# Fashion Recommendation System

This project is a simple AI-based Fashion Recommendation System built using Python.

The system suggests outfits based on weather and occasion entered by the user.

## Features
- Outfit recommendation based on occasion
- Weather-based styling suggestions
- Simple AI logic using Python

## Technologies Used
- Python

## How it Works
1. User enters weather (summer / winter / rainy)
2. User enters occasion (casual / party / office)
3. The system suggests a suitable outfit

## Example

Input:
Weather: summer  
Occasion: party

Output:
Recommended Outfit: Light Summer Dress
